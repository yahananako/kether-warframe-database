# Discord Profile Banner Theme

## 目標

讓 /profile 的 Discord 名片更接近 Discord 個人名片。

## 新增資料

- Discord banner hash
- Discord accent_color
- Discord avatar decoration asset
- Discord nameplate palette

## 顯示效果

- 有 Discord banner 時顯示版圖
- 沒有 banner 時使用 accent_color 與 nameplate palette 產生漸層版圖
- 顯示 Discord 頭像
- 隱藏 Guild ID
- 用 CSS 模擬柔光與玻璃特效

## 限制

Discord Profile Effect 目前不在一般 OAuth User Object 內，因此網站用 CSS 模擬光暈特效，不做 1:1 Profile Effect 複製。

## 版本

不修改網站版本。
