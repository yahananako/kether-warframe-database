# Official News API Route

## 目標

建立官方新聞摘要的 API 前置路由，為後續 RSS／API 自動更新做準備。

## 新增路由

- `/api/official-news`

## 目前狀態

目前先回傳 `data/officialNews.ts` 內的靜態官方公告看板資料：

- 官方公告看板主文案
- 官方新聞入口
- 更新紀錄入口
- 官方論壇入口

## 後續規劃

下一階段可接入官方新聞來源，讓首頁自動顯示：

- 最新官方新聞標題
- 發布日期
- 新聞摘要
- 官方連結
- 快取更新時間

## 注意

- 不修改網站版本號。
- 不修改首頁畫面。
- 此步驟只建立 API 前置路由。
