# KETHER Warframe Database v2.5.44

## 更新類型

首頁區塊標題圖示化。

## 本版更新

- 移除首頁各區塊標題前方數字
- 將區塊標題改為圖示化標籤
- 數據區、導航區、分類資料狀態、網站資訊區、官方資訊區、資料庫狀態、備註皆套用一致樣式
- 保留首頁既有資料讀取、搜尋、通知、導航與版本狀態功能

## 圖示對應

- 數據區：ClipboardList
- 導航區：Menu
- 分類資料狀態：Layers
- 資料區：Layers
- 網站資訊區：Info
- Warframe 官方資訊區：MessageCircle
- 資料庫狀態：ClipboardList
- 備註：Pencil

## 備註

本版只調整首頁區塊標題視覺，不更動資料讀取邏輯。
